# MedSimplify

A web tool that converts complex medical literature into plain language anyone can understand.

## What it does

Takes dense research papers, clinical studies, and medical abstracts and simplifies them to an 8th-grade reading level while keeping all the important information.

## Features

- Paste text or load example studies
- AI-powered simplification using Claude API
- Shows reading level metrics (before/after)
- Clean, minimal interface

## Built with

- Vanilla JavaScript
- Claude Sonnet 4 API
- HTML/CSS

## How to use

1. Open `medsimplify.html` in a browser
2. Paste medical text or try an example
3. Click "Transform Text"
4. Get simplified output with readability stats

## Why I built this

Medical research should be accessible to everyone, not just specialists. This tool helps bridge that gap.
